A live feed of every non-draft product is generated automatically from your catalog. Submit this URL once to start showing your products in Google Shopping and Instagram/Facebook tagging — it stays in sync as you add or change products.
https://zuwera.store/api/product-feed Preview

Meta Commerce Manager: Catalog → Data sources → Data feed → Scheduled feed → paste URL.
📋 Adding a Brevo backup email key
1. Sign up free at brevo.com (300 emails/day free, no credit card)
2. Go to SMTP & API → API Keys → Create API Key
3. Copy the key and add it to Cloudflare Pages → Settings → Variables & Secrets as BREVO_API_KEY
4. Redeploy — Zuwera will automatically fall back to Brevo if Resend is unavailable

📊 What gets tracked

Page View — opens any page
View Content — opens a product
Add to Cart — adds an item to the bag

Purchase — completes an order (counted 3 ways for accuracy)
Lead — joins the waitlist / newsletter
Complete Registration — creates an account
