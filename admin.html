Loyalty

Signed-in shoppers earn points on every order and swap them for money off. Redeeming mints a single-use coupon that runs through your normal promo path — it never changes how anything is priced. Guests don't earn (there's no account to credit).

Earning

Rewards shoppers can pick

Add as many as you like — shoppers choose whichever they can afford. Bigger tiers usually give better value.

Reward code rules

Refer a friend

Each signed-in shopper gets their own code to share. Their friend gets a discount; when that friend's order comes through, the referrer is paid in ${'points'} automatically. Codes are created the first time someone opens Refer a Friend, so your Coupons list only grows for people who actually share.

Change these any time — saving also updates codes people have already shared, so everyone's on the same terms. Points already paid out aren't clawed back.

Members

Bundles

Group pieces that go together. On any product in a set, shoppers see a “complete the set” row with the other pieces — a simple way to lift order value.

Add an offer by picking one of your coupons — or creating one right here. Either way, saving locks that code to only this set's products, so it can't be spent on anything else. It still lives in Coupons if you want to change the discount later.
